# 人大金仓

人大金仓数据库支持V7和V8两个版本,但两个版本的jdbc驱动不同.

## V7

```text
[
	{
		"url" : "jdbc:kingbase://127.0.0.1:54321/X",
		"username" : "system",
		"password" :"krms",
		"includes": [],
		"excludes": [],
		"enable" : true
	}
]
```

## V8

```text
[
	{
		"url" : "jdbc:kingbase8://127.0.0.1:54321/X",
		"username" : "system",
		"password" :"krms",
		"includes": [],
		"excludes": [],
		"enable" : true
	}
]
```
